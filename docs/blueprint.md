# **App Name**: Octet Insights

## Core Features:

- Service Overview: Display key IT services provided by Octet Systems, each with a brief description and a clear call to action for further engagement.
- Portfolio Display: A list of successfully completed projects (CCTV installations, app development etc.) will be made public, along with descriptions and relevant images. A clear call to action to "request a quote" should be present.
- AI-Powered Tech FAQs: Use a generative AI tool to provide quick answers to common IT questions. For instance, provide information and tutorials on common issues with network setups, security questions or password advice.
- Campaign Overview: Showcase awareness campaign summaries and resources.
- Consultation Booking Link: Provide easy access to booking consultations with experts, highlighting the benefits of such consultations. Users can make calendar bookings using a link.

## Style Guidelines:

- Primary color: Deep navy blue (#1A237E), evoking trust and stability.
- Background color: Very light gray (#F5F5F5) to ensure a clean, modern interface.
- Accent color: Muted teal (#4DB6AC), provides a pop of color without being too overwhelming.
- Body font: 'PT Sans', a modern and readable sans-serif for all text.
- Headline font: 'Space Grotesk', providing a computerized feel.
- Use modern, flat icons related to IT services to guide users. All icons will use the accent color (#4DB6AC).
- Clean and minimalist layout, using white space effectively. Prioritize clear sections and easy navigation to enhance user experience.
